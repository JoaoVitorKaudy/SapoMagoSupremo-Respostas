# Projeto Sapo Mago Supremo

Este é um projeto de chatbot feito em HTML, CSS, JavaScript ePHP. Ele usa o [Ollama] (https://ollama.com/) rodando localmente para gerar as respostas do Sapo Mago.

## Como Rodar o Projeto

Para que este projeto funcione na sua máquina, você precisa de 3 componentes principais: um servidor web (XAMPP), o Ollama e um modelo de linguagem.

### Requisitos
1. XAMPP (com PHP): Necessário para rodar o backend ('api.php')
2. OLLAMA: O motor que gera as respostas. Baixe e instale o Ollama em (https://ollama.com/)
3. UM MODELO DE LINGUAGEM: Após instalar o Ollama, rode 'ollama pull llama3' no terminal.

### Passos para executar
1. Clone ou Baixe este repositório
2. Mova os arquivos do projeto para dentro da pasta 'htdocs' do seu XAMPP
3. Inicie  o Ollama (no linux/macOS ele roda como um serviço, no Windows, abra o app)
4. Inicie  o Apache no painel de controle do XAMPP
5. Acesso o projeto no seu navegador através do endereço localhost
